# Gemini AI Chatbot - Full-Stack Django Project

Modern multimodal AI chatbot built with **Django** + **Google Gemini API** (text + image vision).  
Users can register/login, chat with AI, upload images for analysis, view permanent history, clear chats.

Perfect 3-month internship showcase: full-stack, auth, DB, responsive UI, AI vision!

### Features
- User authentication (Register/Login/Logout)
- Permanent chat history per user
- Image upload + Gemini Vision (ask about photos!)
- Beautiful chat UI with avatar, name, bubbles, auto-scroll
- Clear Chat button
- Responsive & mobile-friendly

### Tech Stack
- Backend: Django
- AI: Google Gemini 2.5-flash (multimodal)
- Frontend: HTML/CSS/JS, Font Awesome
- Database: SQLite
- File Upload: Django ImageField
